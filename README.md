# Django WSGI & ASGI Masterclass

Welcome! This repository is focused on mastering the backend architecture of Django, specifically the differences and implementations of **WSGI** (Synchronous) and **ASGI** (Asynchronous).

## 🚀 What's inside?

- A structured roadmap to move from beginner to advanced backend development.
- Deep dives into production servers like **Gunicorn** and **Uvicorn**.
- Real-time communication using **Django Channels** and **WebSockets**.

## 📂 Project Structure

- [planning.md](planning.md): Your step-by-step learning roadmap.

## 🛠️ Key Technologies

- **Python & Django**: The core framework.
- **WSGI**: Traditional synchronous gateway (Gunicorn/uWSGI).
- **ASGI**: Modern asynchronous gateway (Daphne/Uvicorn).
- **Redis**: Used for channel layers in real-time features.
